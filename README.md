# 💰 Personal Finance Manager

A web-based application developed using Java Full Stack technologies to help users manage their daily income and expenses efficiently.

---

## 📌 Project Description

Personal Finance Manager is a simple and user-friendly application that allows users to:
- Track income and expenses
- View financial summary
- Filter expenses by category
- Manage financial records effectively

The main goal of this project is to improve financial awareness and help users make better financial decisions.

---

## 🚀 Features

- ➕ Add Income
- ➖ Add Expense
- 📊 Dashboard (Total Income, Expense, Balance)
- 🔍 Filter Expenses by Category
- 🗑️ Delete Records
- 📋 View Income & Expense List

---

## 🛠️ Technologies Used

### Frontend:
- HTML
- CSS
- JavaScript

### Backend:
- Java
- Spring Boot

### Database:
- MySQL

---

## ⚙️ How It Works

1. 👤 New users create an account using the **Register** page.
2. 🔐 Existing users log in securely using their credentials.
3. 💰 Users can add income and expense details.
4. 📤 The application sends the data to the Spring Boot backend.
5. 🗄️ Data is stored and managed in the MySQL database.
6. 📊 The dashboard displays:
   - Total Income
   - Total Expenses
   - Current Balance
7. 📋 Users can view, update, and delete their income and expense records.
